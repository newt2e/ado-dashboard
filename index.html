    Generate your Azure DevOps Personal Access Token (PAT)

    1. Open your Personal Access Tokens page: https://dev.azure.com/SmartQ-Global-Digital-Cafeteria-Suite/_usersSettings/tokens
    2. Click Create Token.
    3. Enter a Name, select SmartQ-Global-Digital-Cafeteria-Suite as the Organisation, set Expiration to 30 days or longer. Under Scopes select Custom defined, scroll to Work Items and tick Read only.
    4. Click Create.
    5. Copy the token immediately — it is only shown once.
    6. Paste it into the PAT field in the dashboard.
    ⚠ The token is only shown once. Copy it before closing the ADO window.
